---

## 🛠️ Structura proiectului:
* `index.html` - Pagina principală (Acasă)
* `curiozitati.html` - Istoricul și curiozitățile liceului
* `Olimpiade-si-olimpici.html` - Tabelul cu rezultatele elevilor
* `realizari.html` - Secțiunea dedicată echipei de robotică
* `imagini/` - Folderul care conține toate elementele media (foto)
* Fișiere `.css` separate pentru stilizarea fiecărei pagini în parte.
